수강 노트

1.tag종류

(1)strong/ : 굵은 글씨
(2)h1~6/ : 제목
(3)u/ : 밑줄
(4)p/ : 단락지정
(5)br : 줄바꿈
(6)span/ : 크기변경
(7)img : 이미지등록
(8)ul(비순서),ol(순서)-li/ : 목록,목차
(9)table-tr(행)-td(열) : 표만들기
(10)title/ : 제목명시,검색엔진에노출
(11)a/ : 링크

2.tip

(1)동시입력
(윈도우:컨트롤키+대상위치선택)
(2)한글이 깨질 때
(태그meta charset="utf-8"입력)
(3)관용적 서두 : !doctype html
(4)html-head(title,meta),body

3.속성

(1)단락상단 간격 조정(css) : p style="margin-top:px;"
(2)이미지 속성 :
img src="파일위치" width="px,%" height="px,%"
(3)링크 속성 :
a href="주소" target="_blank" title="툴팁태그"
